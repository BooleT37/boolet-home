# boolet-home
My home page

## scripts:

### for production:
* **`build:all`** - build everything concurrently
* **`build:server`** - build server
* **`build:client`** - build client
* **`start`** - start application
### for developement:
* **`watch:server`** - start server in watch mode
* **`watch:client`** - start webpack in watch mode
